# Research Statement

## AI-assisted Campus Spatial Optimization

### Research Abstract

> This research proposes an AI-assisted design workflow for architectural spatial optimization, integrating **human behavior data analysis**, **generative AI**, and **BIM modeling**. 
>
> By combining pedestrian flow analysis with Stable Diffusion-based concept generation and parametric BIM modeling, the workflow enables data-driven architectural design decisions that are responsive to actual human behavior patterns.

### Research Questions

1. **RQ1:** How can computer vision and sensor data be used to model human behavior patterns in architectural spaces?
2. **RQ2:** To what extent can Stable Diffusion generate architecturally feasible spatial layouts conditioned on behavior data?
3. **RQ3:** How can AI-generated concepts be effectively translated into parametric BIM models?

### Methodology

| Phase | Method | Tools | Expected Output |
|---|---|---|---|
| Data Collection | Video tracking, Wi-Fi positioning | Python, OpenCV | Behavior dataset |
| Analysis | Spatial statistics, flow analysis | Pandas, NumPy, SciPy | Flow patterns, heatmaps |
| Generation | Stable Diffusion + ControlNet | SDXL, ComfyUI | Design concepts |
| Modeling | Parametric BIM generation | Rhino, Grasshopper, Revit | 3D models |
| Evaluation | Spatial metrics + expert review | Python, Revit API | Design scores |

### Expected Contributions

1. A replicable AI-assisted architectural design workflow
2. Open-source tools for behavior-aware spatial design
3. Comparative analysis of AI-generated vs. human-designed layouts
4. Integration framework connecting AI generation with BIM tools
