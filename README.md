# AI-Architecture-Research 🏗️🤖

> AI-assisted design workflow for architectural spatial optimization.

---

## Project Overview

**Objective:** Develop an AI-assisted workflow for architectural space optimization.

### Background

This research explores the integration of artificial intelligence into the architectural design pipeline, focusing on campus spatial optimization based on human behavior data analysis.

## Research Directions

### 1️⃣ Spatial Behavior Analysis
- Pedestrian flow simulation and prediction
- Space usage pattern recognition
- Environmental behavior mapping

### 2️⃣ AI-assisted Generative Design
- Stable Diffusion for architectural concept generation
- ControlNet-based conditioned generation
- Design space exploration with latent diffusion

### 3️⃣ BIM Integration
- Automated model generation from AI output
- Parametric BIM component creation
- Data-driven design optimization

## Getting Started

```bash
# Clone the repository
git clone https://github.com/luisdingww-bit/AI-Architecture-Research.git
cd AI-Architecture-Research

# Install dependencies
pip install -r requirements.txt

# Run data preprocessing
python scripts/data_processing.py

# Generate design proposals
python scripts/ai_generation.py
```
